# LIMS 实验室信息管理系统 工作任务书

---

## 一、项目概述

### 1.1 项目名称
智能实验室信息管理系统（Intelligent LIMS）

### 1.2 项目目标
构建一套综合性的实验室信息管理系统，实现实验室工作流程、资源和数据的高效管理。

---

## 二、项目时间节点计划

## 四、功能模块清单

| 5 | 试验管理 | 试验方法管理、试验数据录入、试验结果管理 |
| 6 | 委外管理 | 委外检测管理、委外进度跟踪 |

## 五、技术方案

### 5.1 技术架构

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React 18 + Vite + TypeScript |
| UI组件库 | Ant Design v5 |
| 后端框架 | Java + Spring Boot |
| 数据库 | MySQL |
| 缓存 | Redis |
| 消息队列 | RabbitMQ |

### 5.2 部署方案

- 支持 Docker 容器化部署
- 支持 Vercel 云端部署（前端）
- 支持本地服务器部署
